## 🔧 快速安裝

### 1. 下載專案
```bash
git clone https://github.com/YOUR_USERNAME/shopee-ai-uploader.git
cd shopee-ai-uploader
```

### 2. 安裝套件
```bash
npm install
```

### 3. 設定 API 金鑰
```bash
cp .env.example .env
# 編輯 .env 填入你的 Shopee API 資訊
```

### 4. 執行 AI 示範
```bash
npm run demo
```

---

## 🎮 三種使用模式

### 1️⃣ 快速模式（推薦新手）
```bash
npm start
# 使用 AI 預設設定，最簡單快速
```

### 2️⃣ 進階模式
```bash
npm run upload:bulk
# 批量上傳，自訂 AI 參數
```

### 3️⃣ 開發模式
```bash
npm run dev
# 即時修改，適合客製化開發
```
